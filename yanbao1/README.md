# yanbao1 – Seorabeol single page site

This folder contains the production-ready single-page website for the high-end Korean restaurant brand "萨拉伯尔 Seorabeol".

To upload it to a GitHub repository named `yanbao1`, initialize a repo in this folder and push:

```bash
git init
git add index.html README.md
git commit -m "Add Seorabeol single page"
git remote add origin git@github.com:<your-account>/yanbao1.git
git push -u origin main
```

The `index.html` file is fully self-contained (HTML + CSS + JS) so it can be deployed as-is (e.g., GitHub Pages).
