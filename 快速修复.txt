================================================================================
                        🔴 错误信息快速修复
================================================================================

你看到的错误信息：

    'docker-compose' 不是内部或外部命令，也不是可运行的程序或批处理文件。

这意味着：❌ Docker 未安装

================================================================================
                        ⚡ 3 步快速修复
================================================================================

第一步：安装 Docker Desktop

  1. 打开浏览器
  2. 访问：https://www.docker.com/products/docker-desktop
  3. 点击 "Download for Windows"
  4. 下载完成后，双击安装
  5. 按照安装向导操作
  6. 重启计算机

第二步：启动 Docker Desktop

  1. 打开 Docker Desktop 应用
  2. 等待启动完成（看到右下角图标变绿）
  3. 这需要 30-60 秒

第三步：重新运行 start.bat

  1. 打开文件管理器
  2. 进入 c:\学生档案资料\编程
  3. 双击 start.bat
  4. 等待 15 秒
  5. 访问 http://localhost

✅ 完成！

================================================================================
                        📋 错误信息详解
================================================================================

错误 1️⃣: 'docker-compose' 不是内部或外部命令

  原因：Docker 未安装或未启动
  
  解决方案：
  1. 安装 Docker Desktop
  2. 启动 Docker Desktop
  3. 重新运行 start.bat

错误 2️⃣: 'null' 不是内部或外部命令

  原因：脚本执行出错
  
  解决方案：
  1. 关闭命令行窗口
  2. 重新打开命令提示符
  3. 重新运行 start.bat

错误 3️⃣: version is obsolete

  原因：docker-compose.yml 版本过时
  
  影响：无（只是警告，不会影响运行）
  
  解决方案（可选）：
  1. 编辑 docker-compose.yml
  2. 删除第一行 version 属性
  3. 保存文件

================================================================================
                        🔍 验证 Docker 是否安装
================================================================================

打开命令提示符，输入以下命令：

  docker --version

如果显示版本号（例如 Docker version 24.0.x），说明已安装。
如果显示 "不是内部或外部命令"，说明未安装。

================================================================================
                        🔍 验证 Docker 是否运行
================================================================================

打开命令提示符，输入以下命令：

  docker ps

如果显示容器列表（可能是空的），说明 Docker 正在运行。
如果显示错误，说明 Docker 未启动。

================================================================================
                        📥 Docker Desktop 下载链接
================================================================================

官方下载地址：
  https://www.docker.com/products/docker-desktop

选择 "Docker Desktop for Windows"

下载大小：约 500MB

系统要求：
  - Windows 10 64-bit: Pro, Enterprise, or Education (Build 19041 or higher)
  - 或 Windows 11 64-bit
  - 至少 4GB RAM

================================================================================
                        ⚙️ Docker Desktop 安装步骤
================================================================================

1. 双击下载的 Docker Desktop Installer.exe

2. 按照安装向导操作

3. 勾选 "Use WSL 2 instead of Hyper-V"（推荐）

4. 点击 "Ok" 开始安装

5. 安装完成后重启计算机

6. 打开 Docker Desktop 应用

7. 等待 Docker 启动（约 30-60 秒）

8. 看到右下角 Docker 图标变绿，说明启动成功

================================================================================

   cd c:\学生档案资料\编程

4. 运行 start.bat
   
   start.bat

5. 等待 15 秒

6. 访问应用
   
   http://localhost

================================================================================
                        🎯 常用命令
================================================================================

查看 Docker 版本：
  docker --version

查看 Docker Compose 版本：
  docker-compose --version

查看正在运行的容器：
  docker ps

查看所有容器（包括已停止的）：
  docker ps -a

查看容器日志：
  docker-compose logs -f

停止所有容器：
  docker-compose down

重启所有容器：
  docker-compose restart
